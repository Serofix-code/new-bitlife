# Getting Started

1. Open `index.html` or the public GitHub Pages website.
2. Create or select a local user account.
3. Create a character and choose identity, country, reproductive role, upbringing, special talent, occult type, portrait, and optional seed.
4. Use the main tabs to manage work, assets, relationships, activities, expansions, and the life journal.
5. Press **Age** to advance one year.

Most actions use one annual activity point. Every age-up refreshes action points, processes money, relationships, careers, mortality, family events, market changes, and a possible story event.

The optional story seed makes random generation reproducible for testing.

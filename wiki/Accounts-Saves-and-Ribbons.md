# Accounts, Saves, Challenges, and Ribbons

Each browser can contain multiple local user accounts. Every user has separate autosaves, named saves, ribbon collections, and one-time challenge completion.

Ribbons are awarded when a life ends and are saved to the account. Challenges can award completion only once per user. JSON exports can move saves between browsers or devices.

The optional home server shares one save between devices on the same private network.
